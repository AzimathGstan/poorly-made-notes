# 拗音 ゃゅょ
2文字の場合、1拍と数える、For hiragana, its paired with i series.
ひゃ ひゅ ひょ

# 撥音 ん
１拍と数える
Never read it as a-n is "a n"

# 促音 っ
1拍と数える
no need to pronounce the small つ

# 長音 
母音+相同母音の場合、連続して発音する、同詞

ぬう=> nu~u
はあ=> ha~a

And
え+い　=> え~え
お+う　=> お~お

Example:
けいき　=> け~え~き
こうこう　=> こ~お~こ~お
or in arrow
こうこう　=> こ$\rightarrow$こ$\rightarrow$
